SaviBot

Le bot multi-fonctions pour MediaWiki

Objectif

SaviBot assiste les contributeurs dans les tâches répétitives et techniques afin d'améliorer la qualité et la cohérence des projets wiki.

Fonctionnalités

  • Gestion et ajout automatique des bandeaux de portails
  • Organisation et maintenance des catégories
  • Vérification et gestion des images
  • Surveillance des modifications récentes
  • Gestion des liens interwiki
  • Correction syntaxique et amélioration du code wiki

Conditions pour devenir développeur SaviBot

Connaissance de Python et Pywikibot, maîtrise de MediaWiki API et bonne compréhension des tâches de maintenance sur les projets wiki.

Technologies utilisées

  • Python
  • Pywikibot
  • JavaScript
  • MediaWiki API
