À propos de SaviBot

SaviBot est un bot multifonctions créé le 14 juillet 2025. Il a pour objectif d’assurer la maintenance et le bon fonctionnement des projets MediaWiki.

Il est développé principalement en Python avec Pywikibot et mwclient, avec des composants en JavaScript pour des tâches simples.

Le projet est évolutif et peut être adapté selon les besoins des communautés.

Développeurs

SaviBot est un projet ouvert. Les contributions sont possibles pour améliorer ses fonctionnalités.

Pour participer, il suffit d’avoir des bases en Python ou JavaScript et une compréhension générale de MediaWiki.
